# Projet-Electronique
Ce projet s’inscrit dans la continuité d’un précédent projet portant sur la motorisation et l’asservissement d’un chariot à l’aide de roues d’hoverboard.
Dans le cadre de ce projet, nous souhaitons permettre le pilotage à distance du ≪ OverTrailer ≫ à l’aide d’une télécommande Radiolink. 

L’objectif est que le robot puisse être commandé à distance tout en assurant un déplacement stable et rectiligne.
